
# TaraTOR

Trustless Anonymous Relay Access on The Onion Router is a modern self-hosted privacy-focused web chat where every byte of data is strictly encapsulated within the Tor network with no centralized gateways.


## Features

- User Authentication and Account Management
- Secure Chat
    - End-to-end encrypted messaging
    - Currently supports one-on-one chats with unique chat IDs
- Chat Interface and Messaging
    - Real-time chat UI
- Connection Sharing and Discovery
    - QR code generation for sharing connection codes allowing easy pairing for chats
    - Friend list


## Tech Stack

**Client:** React + TypeScript + Vite, TailwindCSS and shadcn/ui

**Backend:** Python with FastAPI

**Deployment:** Includes Docker support
